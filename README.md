1️⃣ What is the difference between var, let, and const?

উত্তর:
var, let, এবং const JavaScript-এ variable declare করার জন্য ব্যবহৃত keyword।
var function scope এ কাজ করে এবং একই variable আবার declare করা যায়।
let block scope এ কাজ করে এবং একই variable আবার declare করা যায় না, তবে value পরিবর্তন করা যায়।
const block scope এ কাজ করে এবং declare করার পরে এর value পরিবর্তন করা যায় না।

2️⃣ What is the spread operator (...) ?

উত্তর:
Spread operator (...) ব্যবহার করা হয় array বা object এর element গুলোকে আলাদা করে ছড়িয়ে দেওয়ার জন্য। এটি সাধারণত data copy করা, merge করা বা নতুন data তৈরি করার সময় ব্যবহার করা হয়।

3️⃣ What is the difference between map(), filter(), and forEach()?

উত্তর:
map() array এর প্রতিটি element এর উপর কাজ করে এবং একটি নতুন array return করে।
filter() নির্দিষ্ট condition অনুযায়ী element বেছে নেয় এবং নতুন array return করে।
forEach() array এর প্রতিটি element এর উপর loop চালায় কিন্তু কোনো নতুন array return করে না।

4️⃣ What is an arrow function?

উত্তর:
Arrow function হলো JavaScript-এর একটি সংক্ষিপ্ত (short) function syntax, যা => চিহ্ন ব্যবহার করে লেখা হয় এবং সাধারণ function এর তুলনায় সহজ ও ছোটভাবে function তৈরি করতে সাহায্য করে।

5️⃣ What are template literals?

উত্তর:
Template literals হলো JavaScript-এ string লেখার একটি আধুনিক পদ্ধতি যেখানে backtick ( ) ব্যবহার করা হয়। এর মাধ্যমে সহজে string এর ভিতরে variable বা expression ব্যবহার করা যায় এবং multi-line string লেখা যায়।
